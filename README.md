## Create a todo list

```
Let's create a todo list. The tasks file is already created and referenced here.

Requirements

- The initial tasks are available in the /data/tasks.ts file
- The user should be able to create a new task and add it to the list
- The user should be able to remove a task from the list. You must use an icon from the Lucide lib in this button
- The user should be able to mark a task as completed. You must use an icon from the Lucide lib, as well
- You must use the Indigo color theme from Tailwind CSS in dark mode
- It should have a status badge
- The completed date, when available, should be displayed in a tooltip when the person hovers the status badge
```
